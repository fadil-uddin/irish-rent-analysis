# Irish Rental Affordability Analysis

This project analyses Irish rental market data to identify regional rent trends, affordability pressure, and practical housing-market patterns.

The goal is to produce a compact data-analysis and consulting-style project using public rental data, Python, pandas, and clear visual communication.

## Research Question

How have rents changed across Irish regions over time, and what do the patterns suggest about regional affordability pressure?

## Planned Outputs

- Cleaned rental dataset
- Data cleaning notebook
- Exploratory analysis notebook
- 6--8 charts
- 1-page executive summary
- 5--7 slide consulting deck
- Final README with findings and limitations

## Tools

- Python
- pandas
- NumPy
- matplotlib
- Jupyter Notebook
